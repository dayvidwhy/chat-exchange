## Install
Clone the repository and start development mode.
```bash
git clone git@github.com:dayvidwhy/chat-exchange.git
cd chat-exchange
npm i
npm start
```

Project uses NVM for [node](https://github.com/nvm-sh/nvm) version. 
```bash
# Set node to 21
nvm use
```
